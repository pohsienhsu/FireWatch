Clustering Analysis 






Brief: Additional yearly clusterings analysis shown here over the grouping between fire size and average AQI.

Input Features: 

FIRE_SIZE --> Size of each fire considered 
AVG_AQI --> AQI level averaged over the entire duration of the fire
CONT_TIME --> date the fire was contained, suppressed
LATITUDE, LONGITUDE --> location variables
DAYS --> length of fire
    	  		
Output Features: 2 PCA Components

Types: Histogram, Line Graph, Scatterplots

Packages: sklearn (i.e. cluster, preprocessing, KMeans, PCA), seaborn, matplotlib, numpy, pandas

Train/Test Sets: These are unsupervised learning algorithms, ergo the labels were "created" from a syncretizing of our fires and aqi datasets.

Description: PCA/K-Means analysis was performed on roughly 2500 fires for each year over the range of dates (i.e. 1992-2015). Here we see a histogram of the explained variance for the 2 components and the elbow method decision (indicating K=3 elbow) for one of the years, 2006. This is one instance of dimensionality reduction in our work. Many features were converted into mroe informative z-space PCA vectors.
    		






Description (cont.): The PCA components were used as input to a K-Means analysis; yearly clusterings shown here. Roughly 2500 fires across the range of dates (i.e. 1992-2015) were selected for this analysis. These incorporate more features (as explained earlier) and so provide insight on the similarity of each fire using more data. We see a more stable grouping, as evidenced by the rough equality of the various clusters.
