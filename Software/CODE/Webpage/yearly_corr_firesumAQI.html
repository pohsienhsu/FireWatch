Yearly Correlation (Fire Sum vs. AQI) 


Brief: These analysis plots show a yearly data for a per month correlation between AQI and fire sum. 

Features: FIRE_SUM --> FIRE_SIZE * fire length
		  AQI --> air quality index

Types: Scatterplots

Packages: sklearn, seaborn, matplotlib, numpy, pandas

Description: These analysis plots show a yearly data for a per month correlation between AQI and fire sum. Considering AQI isn’t changing simultaneously with the occurrence of wildfires, we measure the monthly average of AQI with respect to wildfire sum. Wildfire sum equals to fire size * the duration of the fires for better addressing the delay effect of wildfire on AQI. From the parity plots, it is shown that most wildfires have smaller effect on AQI than anticipated. So there should be other "main" factors increasing AQI such as gasoline combustion and factory gas emission.
