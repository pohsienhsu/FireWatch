Monthly Paired Data Analysis (Acres Burned) 


Brief: These analysis plots show the monthly paired data for acres burned across month and average AQI level. 

Features: Fire Size (sum per day), Average AQI, Acres Burned

Types: Line Charts

Packages: sklearn, seaborn, matplotlib, numpy, pandas

Data: Fires Dataset, Years 1998-2015

Description: These analysis plots show the monthly paired data for acres burned across month and average AQI level. Each individual year is shown as a separate line, with an average for each month for both metrics shown as the dotted trend lines. This allows us to observe an advanced pairing between air quality levels, acres burned, and fire size. Note the trend lines show a "peak" mid-year, which corresponds with the likelihood that wildfires have more veracity in the summer hot months.
